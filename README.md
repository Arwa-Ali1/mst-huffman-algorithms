# 📊 MST & Huffman Algorithms Project

This project implements and analyzes two major algorithm topics:
- Minimum Spanning Tree (MST)
- Data Compression using Huffman Coding

## 📌 Overview
The project is divided into two parts:
1. MST Algorithms (Prim & Kruskal)
2. Huffman Coding vs Fixed-Length Encoding

## 🚀 Features

### Part 1: MST Algorithms
- Implementation of Prim’s algorithm
- Implementation of Kruskal’s algorithm
- Runtime comparison between both algorithms
- Graph-based problem solving

### Part 2: Huffman Coding
- Huffman encoding and decoding
- Fixed-length encoding for comparison
- Performance evaluation (compression ratio & runtime)
- File generation and testing (50KB, 200KB, 1MB)

## 🛠️ Technologies Used
- Java
- Data Structures:
  - Priority Queue (Min-Heap)
  - Trees
  - Graphs
  - HashMap
- Algorithms:
  - Greedy Algorithms
  - MST (Prim & Kruskal)
  - Huffman Coding
